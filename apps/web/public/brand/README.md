# Uzbek Ta'lim — Brand assets

Bu papkaga loyiha logosi va brend fayllarini qo'ying.

## Kerakli fayllar

- `logo.png` — asosiy logo (web Header/Auth/Footer’da ishlatiladi)

## Tavsiya

- PNG transparent fon bilan, kengligi ~1200px+ (yoki SVG).
- Agar sizda 2 xil variant bo‘lsa:
  - ikon-only (favicon/app icon)
  - wordmark (ikon + “UZBEK TA’LIM”)

Hozircha web kod `"/brand/logo.png"` faylini kutadi.
